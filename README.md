# 🏗️ Construction CRM - Полнофункциональная система управления строительной организацией

> **Современная, мощная и интуитивно понятная CRM система специально разработанная для строительных компаний с расширенными возможностями управления проектами, персоналом, складом и финансами.**

## 🌟 Ключевые особенности

### 💼 **Управление проектами**
- 📋 Полноценное ведение проектов с бюджетами и сметами
- 👥 Управление командами и назначение ролей
- 📊 Отслеживание прогресса в реальном времени
- 🎯 Канбан-доска для управления задачами
- 📅 Интегрированный календарь с планированием ресурсов
- 🌦️ Учет погодных условий для планирования работ

### 🏭 **Складской учет**
- 📦 Многоуровневый учет материалов с QR-кодами
- 🔧 Управление инструментами и оборудованием
- 📱 Мобильное сканирование QR-кодов
- 📋 Автоматические уведомления о низких запасах
- 🚚 Интеграция с поставщиками и заказы
- 📍 Точная локализация (зоны, полки)

### 👨‍💼 **HR и персонал**
- 👤 Расширенные профили сотрудников с сертификациями
- ⏰ Учет рабочего времени с GPS-отметками
- 🎓 Система обучения и развития персонала
- 💰 Автоматический расчет заработной платы
- 🏥 Отпуска и больничные с электронным документооборотом

### 🛡️ **Безопасность и качество**
- 🚨 Система мониторинга инцидентов безопасности
- 📋 Чек-листы качества и проверки
- 📊 Индекс безопасности и аналитика
- 📸 Фотодокументирование процессов
- 🎯 Обучение технике безопасности

### 💰 **Финансовое управление**

## 🚀 Технологический стек

### **Frontend**
- ⚛️ **React 18** с TypeScript для типобезопасности
- ⚡ **Vite** для быстрой разработки и сборки
- 🎨 **Material-UI + Tailwind CSS** для современного дизайна
- 📱 **PWA** для мобильной адаптации и офлайн работы

### **Состояние и данные**
- 🐻 **Zustand** с middleware (persist, devtools, immer)
- 🔄 **React Query** для кэширования и синхронизации данных
- 📝 **React Hook Form + Zod** для валидации форм

### **Графики и визуализация**
- 📊 **Recharts + Chart.js** для интерактивных графиков
- 📈 **Аналитические дашборды** с real-time данными
- 🎯 **KPI виджеты** с динамическими показателями

### **Дополнительные возможности**
- 📱 **QR-сканер** для инвентарного учета
- 🌤️ **Виджет погоды** с рекомендациями для работ
- 📄 **Экспорт в Excel/PDF** с настраиваемыми шаблонами
- 🔒 **Система безопасности** с аудитом действий
- 🔗 **API интеграции** с внешними системами

## 🛠️ Установка и запуск

### Предварительные требования
- Node.js 18+ 
- npm или yarn
- Git

### Быстрый старт

```bash
# Клонирование репозитория
git clone https://github.com/your-username/construction-crm.git
cd construction-crm

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предварительный просмотр сборки
npm run preview
```

```
│   ├── 📁 utils/              # Вспомогательные функции

├── 📄 vite.config.ts         # Конфигурация Vite + PWA
└── 📄 tailwind.config.js     # Настройка Tailwind CSS
```
